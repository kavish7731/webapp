Laundry Wallah / Laundry Mart

A simple HTML webpage for a laundry service business named Laundry Wallah / Laundry Mart. This project demonstrates basic HTML elements such as headings, paragraphs, images, lists, tables, forms, and links.

Features
Welcome section with business introduction
Laundry service image
List of available services
Price list displayed in a table
Booking form for customers
Contact information with email link
Technologies Used
HTML5
